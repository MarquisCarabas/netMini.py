# NetMini - Network Port Scan Detection Tool

A lightweight Python-based network monitoring tool that detects and alerts on potential port scanning activities in real-time.

## Features

- Real-time packet capture and analysis
- Port scan detection using configurable thresholds
- Color-coded alerts for security events
- Automatic logging of detected scans
- Cross-platform support (Windows/Linux)

## Requirements

- Python 3.7+
- Scapy
- Colorama (optional, for colored output)
- Npcap (Windows) or libpcap (Linux)
